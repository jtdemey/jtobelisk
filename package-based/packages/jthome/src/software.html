Ncryptor

Ncryptor is a simple web app used to invoke the GNU Privacy Guard (gpg). It's an easy way to encrypt confidential messages to others, decrypt messages to you, and manage your Ed25519 and RSA keypairs.

KeyringBot

KeyringBot is a bot you can self-host for Discord that stores and retrieves public cryptography keys so you can easily encrypt and decrypt private messages.
